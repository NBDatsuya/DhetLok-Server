# 系统是怎样炼成的

## 1. 需求分析
做一个音乐网站
- 普通用户可以登录注册、查看音乐、收藏音乐、查看歌曲分类、歌手分类以及排名等。
- 管理员可以对歌曲以及歌手进行管理。
  
  功能性需求-》非功能性需求 淘宝秒杀 金融产品低时延 游戏1秒启动
